# goit-pycore-hw-06

У цьому завданні реалізовано консольного асистента для керування адресною книгою контактів.

# Основна функціональність

Асистент дозволяє:

- додавати нові контакти;
- додавати декілька телефонів до одного контакту;
- переглядати всі контакти;
- шукати телефони контакту за іменем;
- редагувати номер телефону;
- завершує роботу програми командою `exit`.

# Команди асистента

| Команда | Опис |
|---|---|
| `hello` | Привітання з асистентом |
| `add name phone` | Додати контакт або телефон до існуючого контакту |
| `change name old_phone new_phone` | Змінити старий номер телефону на новий |
| `phone name` | Показати телефони контакту |
| `all` | Показати всі контакти |
| `exit` | Завершити роботу |

# Приклади використання

hello
add Misha 1234567890
